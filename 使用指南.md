# 🐑 wool-monitor 使用指南（小白流）

> 給未來忘光光的自己。不講技術，只講「怎麼開、每顆按鈕在幹嘛、東西不見了怎麼辦」。

## 一、怎麼開

| 我想要 | 做法 |
|---|---|
| 桌面小浮窗（最常用） | 雙擊 **`floating-widget.bat`** |
| 瀏覽器完整版 | 雙擊 **`start.bat`**，然後開 http://127.0.0.1:3789 |
| 迷你單行版 | 雙擊 `floating-widget-mini.bat`（↑/↓ 換提供商） |
| 關掉浮窗 | 滑鼠移到浮窗右下角，按控制列的 **✖**（或點一下浮窗按 Alt+F4） |

浮窗會自己把伺服器拉起來，不用先跑 `start.bat`。

## 二、浮窗操作

- **搬位置**：按住空白處直接拖。
- **調大小**：拖四個角落（角落有小方塊）。
- **收納到右邊**：空白處**雙擊**，或用下面控制列的 ⏴。
  收起後螢幕右緣會留一條會發光的小把手，滑鼠靠過去自動滑出來，移開幾秒又縮回去。
  要固定展開：再雙擊一次，或按 📌。
- **控制列**：滑鼠移到浮窗**右下角**會浮現一排小工具：

  | 按鈕 | 功能 |
  |---|---|
  | ◐ 滑桿 | 「平常」的卡片背景透明度 |
  | 🖱 滑桿 | 「滑鼠移上來時」的透明度（沒調過就跟 ◐ 一樣） |
  | 👻 | **滑鼠穿透**：點擊直接穿過浮窗、操作它底下的視窗 |
  | ⏴ / 📌 | 收納到右緣 / 取消收納 |
  | ✖ | **關閉浮窗**（伺服器照跑；要再開就雙擊 `floating-widget.bat`） |

- **👻 穿透開了之後浮窗就點不到了（正常！）**，要關掉請按 **`Ctrl+Alt+W`**
  （全域快捷鍵，隨時按都有效，也可以當開關切換用）。

## 三、血條在說什麼

由上往下：

1. **目前使用率** — 5 小時 session 額度，跑 Claude Code / 網頁聊天都算。
2. **下次重生** — 這個 session 視窗什麼時候重置。
3. **CONTEXT 使用率** — **本機**最近 30 分鐘內活躍的 Claude Code 對話,
   它的上下文（context window）還剩多少。標籤是那個對話的專案資料夾名。
   ⚠️ 這條跟官網配額無關，也不一定是你「現在正在用」的那個專案——它抓的是全機最近動過的 session。
4. **每週使用率 / 每週重生** — 每週總額度。
5. **Fable 每週使用率** — Fable 模型專屬的週配額（上游哪天拿掉就自動消失）。

顏色：綠 → 70% 變黃 → 85% 變橘 → 92% 變紅（開始閃）。
任一條到 90% 和 95% 時各跳一次 Windows 通知（之後不再吵，等下個重置週期才重新警戒）。

**「未啟動／發訊息後起算」**：官方視窗還沒開始計（跟 claude.ai 顯示
"Starts when a message is sent" 同一個意思），發一則訊息後就會開始倒數，不是壞掉。

## 四、🐑 薅羊毛進度表

估算 Fable 燒掉多少贈送 credit（**台幣顯示**，佛系粗估、不是帳單；
只算本機 log，網頁端聊天算不到）。

表單可以設定：**額度**（美元，預設 $10 贈送 credit）、**匯率**（預設 30）、
**起算日 / 到期日**（留空 = 算全部歷史）。改完按「儲存」。

- 😌 = 到期前燒得完；🚨 = 燒不完，加緊薅！

## 五、東西不見了 / 怪怪的

| 症狀 | 解法 |
|---|---|
| 浮窗不見了 | 先看**螢幕右緣**有沒有發光小把手（被收納了）；沒有就重跑 `floating-widget.bat` |
| 浮窗點不到 | 👻 穿透開著，按 `Ctrl+Alt+W` |
| 數字跟官網對不上 | 官網打開 Settings→Usage 對照；「未啟動」見上面第三節 |
| 變英文了 | 右上角語言鈕（瀏覽器版）點回「中文」 |
| 整個沒反應 | 工作管理員砍掉 `node.exe`,重跑 `floating-widget.bat` |
| 換了新電腦 DLL 缺失 | 見 README「快速開始」的 WebView2 三顆 DLL 抓法 |

## 六、檔案在哪

| 檔案 | 內容 |
|---|---|
| `config.json` | 羊毛表設定（額度/匯率/日期）——不進 git |
| `wool-cache.json` | 羊毛掃描快取，刪掉會重掃而已 |
| 瀏覽器/浮窗各自的 localStorage | 透明度、收納、穿透、語言等偏好（兩邊分開記） |
| `PROGRESS.md` | 給接手的 Claude 看的進度快照 |
